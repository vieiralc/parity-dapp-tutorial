# Voting dapp.

![alt text](https://github.com/vieiralc/votingdapp/blob/master/img/Capture.PNG)

Requirements:

<ul>
  <li>Parity v1.10.4</li>
  <li>NPM</li>
</ul>

To install:

```
$ ./init.sh
$ npm install -g webpack
```

To build:

```
webpack --watch
```

Files will be built into `dist/`. Just symlink that dir into your dapps path.
